# Calculadora de Notas

Uma pequena calculadora acadêmica construída em Python com foco em TDD (Test-Driven Development).  

## Funcionalidades

Cálculo de média

Determinação da situação do aluno:
    "aprovado"
    "recuperacao"
    "reprovado"

Validação de notas

## Regras da situação do aluno

    Média maior ou igual a 7.0 → `aprovado`
    Média entre 4.0 e 6.9 → `recuperacao`
    Média menor que 4.0 → `reprovado`

## Conceitos aplicados

    TDD (Red → Green → Refactor)
    Controle de versão com Git
    Integração contínua (CI)
